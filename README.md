# Radio Ruben — Cast Web Receiver

Custom Cast receiver for the Radio Ruben iOS app.
- URL: https://rubenoid.github.io/cast-receiver/

Plays the station's Icecast AAC stream and renders a branded now-playing screen,
updated per-song over the `urn:x-cast:com.radioruben.np` custom channel.
No secrets — this is a public static page.
